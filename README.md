# Regional Socio-Economic Analysis of England and Wales

## Overview

This project explores socio-economic disparities across **174 local authorities in England and Wales** using UK Census data. The analysis investigates how **economic activity, age structure, commuting behaviour, and vehicle ownership** interact to shape regional socio-economic conditions.

The goal is to identify meaningful regional patterns that can support policymakers, urban planners, transport authorities, and local governments in understanding inequality, mobility, and access to opportunities.
